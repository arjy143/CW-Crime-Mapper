User Guide

Welcome to the crime mapper tool! In this guide I will be explaining the features of this app and how to use it.

Start - MapsActivity

Upon starting the app for the first time, you will be asked to give location and notification permissions. Without these the app will not work. After this, you will need to reload the app.


After the app has been set up for the first time, upon accessing the app for the second time, map markers will now appear. The red marker indicates the current user location or default location. The orange markers indicate where a crime has happened.


If you tap a marker, an info window will appear, showing some basic details about what happened there. If you hold down the marker, the app will take you to the InfoActivity, where you can see more details about that event.


There is a settings button on the top left corner of the app, which will take you to the settings page.


Info - InfoActivity

The InfoActivity is where a user can see more details about the crime that they have tapped on. You are able to see the type of crime, the approximate location of the crime as well as the exact latitude and longitude co-ordinates.


Settings - SettingsActivity

Tapping the settings button on the MapsActivity will take you to the SettingsActivity. This is where a user can be taken to the user guide, the data API or refresh the page.


API button - This takes you to the UK police API, where you can see the documentation for accessing the various data records stored there.


Refresh button - This activates an alarm which sends a broadcast, causing the app to refresh.


Notifications

A notification will appear upon starting the app. This notification will contain the current location of the user and the area that the crimes are being shown in.
